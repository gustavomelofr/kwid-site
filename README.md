# Renault Kwid - Landing Page Premium

Landing page desenvolvida para o Renault Kwid, focada em design moderno, interatividade e experiência do usuário.

## 🚀 Funcionalidades
- **Design Premium**: Dark mode com estética glassmorphism e cores da marca.
- **Carrossel Interativo**: Galeria de fotos de usuários com navegação suave e visualização íntegra (sem cortes).
- **Animações Fluidas**: Revelação de elementos ao rolar a página e navegação suave (smooth scroll).
- **Responsividade**: Site totalmente adaptável para dispositivos móveis e desktop.

## 🛠️ Tecnologias
- HTML5 Semântico
- CSS3 (Variáveis, Flexbox, Grid)
- JavaScript Vanilla (Interatividade e Carrossel)

## 📸 Imagens
As imagens utilizadas no projeto foram fornecidas pelo usuário e capturadas de fontes oficiais para demonstração.
